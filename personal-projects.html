Robotics Simulations

Autonomous Pitch Curator

A high-fidelity MATLAB simulation of an autonomous rover for precision cricket pitch maintenance. Features stochastic defect detection, coverage-optimized Boustrophedon path planning, and a smart "inspect-and-repair" protocol.

FloraBot: Bio-Mimetic Agent

A high-fidelity MATLAB simulation of a robotic plant agent that uses Affective Computing to communicate biological needs. Features species-specific metabolic modeling (Snake Plant vs. Peace Lily), autonomous self-charging logic, and a priority-based FSM that simulates survival instincts.

View Code →

BeachCrab

MATLAB-based simulation of an autonomous beach-cleaning robot. Features a Klann Linkage mechanism for locomotion, reactive obstacle avoidance, and a 4-bar linkage scoop for debris collection.
